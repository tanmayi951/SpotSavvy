# SpotSavy Smart Parking Frontend

A React + Vite frontend for a smart parking dashboard. It simulates real-time parking availability and provides a simple "Guide me to next spot" interaction.

## Run locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Open the URL shown by Vite in your browser.

## What is included

- `src/App.jsx` — dashboard layout with live status cards and spot list
- `src/styles.css` — modern UI styling
- `index.html`, `src/main.jsx` — Vite entry point
- simulated parking data and dynamic state updates
